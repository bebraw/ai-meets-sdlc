# AI meets SDLC – seminar website

This repository contains the website for the AI meets SDLC seminar held 13th of October at Marsio Saastamoinen Foundation Stage, Espoo, Finland.

## Interest List

The interest form stores submissions in Cloudflare D1. Email, name, and organization are encrypted before insert, and a keyed email hash is stored for deduplication.

See [Cloudflare setup](docs/cloudflare.md) for provisioning, local testing, backup, and deployment notes.
